# analisys_tool_dash
seaborn, plotly, sklearn에서 제공하는 데이터와 온라인상의 csv 파일을 웹으로 분석하기 쉽도록 도와주는 웹 서비스입니다.

## Prerequisite
- numpy
- pandas
- matplotlib
- seaborn
- sklearn
- plotly
- dash

## How to start
```shell
$ git clone sajacaros/analisys_tool_dash
$ cd analisys_tool_dash
```

## Installation
```shell
$ pip install -r requirements.txt
```

## Run
```shell 
$ python analysis.py
```
